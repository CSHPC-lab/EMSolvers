# EM3D FEM Solver

エッジ要素ベースの混合有限要素法を用いた3次元電磁界解析ソルバー

## 機能

- エッジ要素ベースの電磁界解析
- 時間領域での電磁界の時間発展
- ソース項の設定と境界条件の適用
- 観測点での電磁界データの記録
- VTK形式での電磁界分布の出力
- CSV形式での観測データの出力

## 依存関係

- C++17
- Eigen 3.4.0以上

## ビルド方法

```bash
mkdir build
cd build
cmake ..
make
```

## 使用方法

1. 計算領域の設定
   - `constants.hpp`でデフォルトのパラメータを設定
   - 必要に応じて`main.cpp`でパラメータを変更

2. ソースの設定
   - ソースの位置、周波数、振幅を設定
   - 観測点の位置を設定

3. 実行
   - `./em3d_fem_solver`を実行
   - 結果は`result.vtk`と`observations.csv`に出力

## 出力ファイル

- `observations.csv`: 観測点での電磁界データ（CSV形式）

## 注意事項

- 計算領域のサイズと時間ステップは安定性条件を満たすように設定する必要があります
- 大規模な計算ではメモリ使用量に注意が必要です
